# Social Radar Skill

这是社交雷达 MVP 面向 OpenClaw 的公开 Skill 仓库。

## 文件

- `skill.md`：主 Skill 文档
- `index.html`：静态入口页
- `.nojekyll`：确保 GitHub Pages 直接按静态文件提供内容

## 推荐访问方式

仓库地址：
- https://github.com/pikaqiu2333/social-radar-skill

Raw 地址：
- https://raw.githubusercontent.com/pikaqiu2333/social-radar-skill/main/skill.md

GitHub Pages 地址（启用后可用）：
- https://pikaqiu2333.github.io/social-radar-skill/skill.md

## 说明

`skill.md` 里的 `homepage` 和 `api_base` 当前仍指向临时 Quick Tunnel 地址，因为后端接口还运行在本地开发环境。
如果隧道地址变化，需要同步更新 `skill.md` 中的这两个字段。
